Aegis

The Aegis module is the main module, and it provides three main functions.

readPost()

Check's if information sent via POST is set, validates it and returns an associative array with the validated data.

Parameters: String (Name of the inputs separated with a comma)

Returns: Array/Boolean

Example: (Assuming we have a form with a name, email and password inputs)

$_->readPost("name,email,password"); Result: ["name"=>"Aegis", "email"=>"email@email.com", "password"=>"hjaos"]
readGet()

Check's if information sent via GET is set, validates it and returns an associative array with the validated data.

Parameters: String (Name of the inputs separated with a comma)

Returns: Array/Boolean

Example: (Assuming we have a form with a name, email and password inputs)

$_->readGet("numer,id"); Result: ["number"=>1, "id"=>132]
p()

This function is a safe echo(), it will encode all HTML entities. The strings printed by this function are the ones declared on the String resources. Android programmers will find this familiar.

Parameters: String

Returns: None

Example:

$_->p("@string/Title"); Result: Echoes the "Title" string
getRoute()

This function get's the current URL and separates it into an array. This is used for the File Router in the index.php.

Parameters: None

Returns: Array

Example:

$_->getRoute(); Result: Returns array separated by the "/".

Security

The security module takes care of validating data and getting information about the user for security purposes.

secureData()

Checks if the data doesn't contain dangerous strings or known spam.

Parameters: String

Optional Parameters: Boolean (Allow HTML in text), Boolean (Allow text to be empty)

Returns: String/Boolean

Example:

$_->security->secureData("Hi there how Are you?"); Result: "Hi there how Are you?" $_->security->secureData("INSERT INTO Database..."); Result: false $_->security->secureData("", false, false); Result: false
getUserIp()

Gets the user's IP adress.

Parameters: None

Returns: String

Example:

$_->security->getUserIp(); Result: "1.2.3.4"
getUserAgent()

Gets the user's User Agent.

Parameters: None

Returns: String

Example:

$_->security->getUserAgent(); Result: User Agent
getUserAgentInformation()

Gets the user's User Agent and orders the data in an array.

Parameters: None

Returns: Associative Array

Example:

$_->security->getUserAgentInformation(); Result: User Agent Information
validateType()

Validates the given data in the given format.

Formats: String:"string", Integer:"int", Boolean:"boolean", Number:"numeric", Array:"array".

Parameters: Data,String

Returns: Boolean

Example:

$_->security->validateType("hello","int"); Result: false

Text

The text module takes care of text-related tasks and functionalities.

buildText()

Builds a text with the given parameters. (Pretty useful for Blogs)

Parameters: Array (Associative Array Containing The Data), String (Data to be placed, (name of the data in the array)), String (Text Wrapper)

Returns: String

Example:

$_->text->buildText(["Title" => "My Title", "Content" =>"<span>hello world</span>" ],"Title,Content","<h1>@Title</h1><p>@Content</p>"); Result: "<h1>My Title</h1><p><span>hello world</span></p>"
capitalize()

Capitalizes every first letter of all the words in the text.

Parameters: String

Returns: String

Example:

$_->text->capitalize("hello world"); Result: "Hello World"
checkSimilarity()

Checks the percentage of similarity between two strings.

Parameters: String, String

Returns: Number

Example:

$_->text->checkSimilarity("hello world","Hello world"); Result: Percentage
getSuffix()

Gets the suffix of a given phrase in a given text.

Parameters: String, String

Returns: String

Example:

$_->text->getSuffix("hello world","hello "); Result: "world"
getPrefix()

Gets the prefix of a given phrase in a given text.

Parameters: String, String

Returns: String

Example:

$_->text->getPrefix("hello world"," world"); Result: "hello"
getAffixes()

Gets the prefix and suffix of a given phrase in a given text.

Parameters: String, String

Returns: Array

Example:

$_->text->getAffixes("hello to all the world"," to all the "); Result: ["hello", "world"]
getStringBetween()

Gets the text between other two strings.

Parameters: String, String, String

Returns: String

Example:

$_->text->getAStringBetween("hello to all the world","hello","world"); Result: " to all the "
removeTags()

Removes all php and html tags from a text.

Parameters: String

Returns: String

Example:

$_->text->removeTags("<span>hello world</span>"); Result: "hello world"
removeSpecialCharacters()

Removes the following characters from a text: #,:,ñ,í,ó,ú,á,é,Í,Ó,Ú,Á,É,(,),¡,¿

Parameters: String

Returns: String

Example:

$_->text->removeSpecialCharacters("Español"); Result: "Espanol"
removePunctuation()

Removes the following characters from a text: ;,.

Parameters: String

Returns: String

Example:

$_->text->removePunctuation("Hey, goodbye."); Result: "Hey goodbye"

Session

The session module handles sessions.

start()

Starts the session, and saves the user's IP and User Agent in an Session variable for security.

Parameters: None

Returns: None

Example:

$_->session->start(); Result: $_SESSION["meta"]=["ip"=>"User's IP","agent"=>"User Agent"];
regenerate()

Regenerates the session ID.

Parameters: None

Returns: None

Example:

$_->session->regenerate(); Result: A new Session ID
end()

Ends the session and unsets all the variables and cookies.

Parameters: None

Returns: None

Example:

$_->session->end(); Result: Session end
check()

Checks if the current user's data matches with the session saved data (IP and UserAgent).

Parameters: None

Returns: Boolean (If current data matches the saved data.)

Example:

$_->session->check(); Result: true/false

Files

The files module handles uploads, files validation, renaming and resize.

getSize()

Gets the size in bytes of a given file.

Parameters: File

Returns: String

Example:

$_->files->getSize(file); Result: Size of the file in bytes.
exists()

Checks if a file already exists.

Parameters: String (File Path)

Returns: Boolean

Example:

$_->files->exists("img/s.jpg"); Result:true/false
delete()

Deletes a File.

Parameters: String (File Path)

Returns: Boolean

Example:

$_->files->delete("img/s.jpg"); Result: true/false
rename()

Renames a File.

Parameters: String (File Path), String (New File Path)

Returns: Boolean

Example:

$_->files->delete("img/s.jpg", "img/side.jpg"); Result: true/false
upload()

Uploads a file.

Parameters: File (Via POST),String (Path to save it to), Number (Allowed Size)

Returns: Boolean

Example:

$_->files->upload(file,"uploads/",20000); Result: true/false

Util

The util module handles some specific tasks.

getRandomString()

Gets a random string of the given length.

Parameters: Integer

Returns: String

Example:

$_->util->getRandomString(5); Result: A 5 length random String

Crypt

The crypt module handles text encryption using Mcrypt and the AES algorithm.

generateKey()

Generates the key for encryption.

Parameters: None

Returns: String

Example:

$_->crypt->generateKey(); Result: A Key for encryption
regenerateKey()

Regenerates the key for encryption.

Parameters: None

Returns: None

Example:

$_->crypt->regenerateKey(); Result: Changes the Encryption Key
setKey()

Sets the key for encryption.

Parameters: String

Returns: None

Example:

$_->crypt->setKey(key); Result: Sets the Encryption Key
getKey()

Gets the key for encryption.

Parameters: None

Returns: None

Example:

$_->crypt->getKey(); Result: Returns the Encryption Key
encrypt()

Encrypts a given text.

Parameters: String (Text to Encrypt)

Returns: String

Example:

$_->crypt->encrypt("text"); Result: "Encrypted Text"
decrypt()

Decrypts a given text.

Parameters: String (Text to Decrypt)

Returns: String

Example:

$_->crypt->encrypt("Encrypted Text"); Result: "Text"

Images

The image module handles Images Operations.

uploadImages()

Uploads a single or multiple images.

Parameters: File (Uploaded Via POST), String (Directory to save to)

Optional Parameters: Number (Max File Size), String (Rename), Number (Width), Number (Height)

Returns: File Path / False if Failed

Example:

$_->util->uploadImages(file, "img/",20000,"new",500,500); Result: Store the Image in img/, change the name to "new" and resize it to 500 x 500 px
