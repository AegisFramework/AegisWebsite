Text

The text module takes care of text-related tasks and functionalities.

capitalize()

Capitalizes every first letter of all the words in the text.

Parameters: String

Returns: String

Example:

$_.text.capitalize("hello world"); Result: "Hello World"
getSuffix()

Gets the suffix of a given phrase in a given text.

Parameters: String, String

Returns: String

Example:

$_.text.getSuffix("This is cool.","This is "); Result: "cool."
getPrefix()

Gets the prefix of a given phrase in a given text.

Parameters: String, String

Returns: String

Example:

$_.text.getSuffix("This is cool.","cool."); Result: "This is "
getAffixes()

Gets the prefix and suffix of a given phrase in a given text.

Parameters: String, String

Returns: JSON

Example:

$_.text.getSuffix("Hello there, how are you?."," how "); Result: {prefix: "Hello there,", suffix: "are you?"}
getSelected()

Gets the currently selected text.

Parameters: None

Returns: String

Example:

$_.text.getSelected(); Result: "User selected text"

Speech

The speech module uses the new Speech Synthesis API and may not work in some browsers.

talk()

Speaks the given text.

Parameters: String (Text), String (Language)

Returns: None

Example:

$_.speech.talk("Hi there!", "en-US"); Result: Audio output of "Hi there!"

Storage

The storage module uses the HTML5 Local Storage and deals with the related tasks.

set()

Stores a given data in a given key-name.

Parameters: String, String

Returns: none

Example:

$_.storage.set("Color","Black"); Result: Stores "Black" in "Color"
get()

Gets the data of a given key.

Parameters: String

Returns: String

Example:

$_.storage.get("Color"); Result: "Black"
clear()

Deletes all the Locally Stored Data.

Parameters: None

Returns: None

Example:

$_.storage.clear(); Result: Deletes all the stored Data

Notification

The notification module uses the new HTML5 notifications API and may not work in some browsers.

requestPermission()

Requests the user permission to show notifications.

Parameters: None

Returns: None

Example:

$_.notification.requestPermission(); Result: The browser will request the permission to the user
show()

Shows a notification.

Parameters: String (Title), String (Text), String (URL of Image)

Optional Parameters: Function (Function to run on the click event), Function (Function to run on the error event), Function (Function to run on the show event), Function (Function to run on the close event)

Returns: None

Example:

$_.notification.show("Hi!", "Hi there!", "http://aegisframework.com/img/notification.png"); Result: A notification will be shown to the user

Screen

The screen module is used to get information about the user's display or device.

isRetina()

Checks if the display has Retina resolution.

Parameters: None

Returns: Boolean

Example:

$_.screen.isRetina(); Result: true/false
isPortrait()

Checks if the device is in Portrait orientation.

Parameters: None

Returns: Boolean

Example:

$_.screen.isPortrait(); Result: true/false
isLandscape()

Checks if the device is in Landscape orientation.

Parameters: None

Returns: Boolean

Example:

$_.screen.isLandscape(); Result: true/false
getOrientation()

Gets the device current orientation.

Parameters: None

Returns: String

Example:

$_.screen.getOrientation(); Result: "portrait"/"landscape"
getMaxiumWidth()

Gets the display width.

Parameters: None

Returns: Number

Example:

$_.screen.getMaxiumWidth(); Result: Number in pixels
getMaxiumHeight()

Gets the display height.

Parameters: None

Returns: Number

Example:

$_.screen.getMaxiumHeight(); Result: Number in pixels
